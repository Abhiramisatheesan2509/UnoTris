# UnoTris

UnoTris is a **Tetris game implemented on Arduino** using an **LED matrix display**.

## Features

- Classic Tetris gameplay
- Piece movement and rotation
- Line clearing
- LED matrix display rendering
- Runs on Arduino

## Hardware Required

- Arduino Uno
- MAX7219 LED Matrix
- Push buttons / joystick for controls
- Jumper wires

## Libraries Used

- LedControl library

## File

- `bot_code.ino` – Main Arduino code for the Tetris game.

## How to Run

1. Connect the LED matrix to the Arduino.
2. Install the **LedControl** library in Arduino IDE.
3. Upload `bot_code.ino` to the Arduino.
4. Use buttons/joystick to control the pieces.
